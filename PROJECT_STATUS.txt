‚ö†Ô∏è  COMMON ISSUES & FIXES:
================================================================================

Issue: MongoDB connection timeout
Fix: Whitelist your IP at https://mongodb.com/cloud/atlas/network-access

Issue: Email not sending
Fix: Add EMAIL_USER and EMAIL_PASS to backend/.env
     Use 16-character Gmail App Password, not regular password

Issue: 401 Unauthorized on API calls
Fix: Your access token expired
    Use refresh endpoint to get new token
    Or login again

Issue: Google login returns 501
Fix: Add Firebase service account JSON to backend/firebase-service-account.json
    Or set FIREBASE_SERVICE_ACCOUNT_PATH env variable

Issue: CORS error
Fix: Check frontend VITE_API_BASE_URL matches backend URL
    Usually should be: http://localhost:5000/api

Issue: Port 5000 already in use
Fix: Change PORT in backend/.env
    Or kill process using port 5000

Issue: Token refresh not working
Fix: Check that refreshToken is in localStorage
    Check that backend.env has JWT_REFRESH_SECRET
